# bridal-estimate-demo

## 概要

ブライダル事業向け「見積作成システム」の商談デモ用インタラクティブモックアップ。
Excel 手入力に代わる、選択式・プルダウン中心の見積業務フローを体験できます。

既存の基幹システム「BVマネージャー」は置き換えず、見積作成に特化した補助システムとして提案するイメージで使用します。

## 実演フロー

1. 見積一覧を確認する
2. 新規見積作成（3ステップウィザード）を開始する
3. 顧客情報・会場・人数・オプションを入力する
4. 見積編集画面で明細を調整する（金額リアルタイム更新）
5. 顧客向け見積書プレビューを確認する
6. PDF出力モーダルを操作する
7. BVマネージャー連携準備画面を確認する

## 画面構成

| パス | 画面 |
|------|------|
| `/` | 見積一覧 |
| `/new` | 新規見積作成（ウィザード） |
| `/edit` | 見積編集 |
| `/preview` | 顧客向け見積書プレビュー |
| `/bv` | BVマネージャー連携イメージ |

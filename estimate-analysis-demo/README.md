# AI見積分析アシスタント（estimate-analysis-demo）

印刷会社向けの **AI見積分析デモアプリ** です。
過去の見積データ・担当者別の価格傾向・同顧客の価格履歴・類似案件をもとに、
**標準価格** と **推奨提示価格** を提示し、見積業務の標準化と属人化解消を支援します。

> ⚠️ 本デモで使用しているデータはすべて **架空データ** です。
> 実在する顧客名・社名・担当者名・案件名・実データは一切含まれていません。

---

## 目的・想定ユースケース

印刷会社が抱える以下の課題を解決するイメージを示すデモです。

- 年間5,400件以上の見積作成があり、営業担当者の業務負荷が高い
- 営業担当者6名によって価格設定にばらつきがある
- 過去25年分の見積データが蓄積されているが、十分に活用できていない
- 同じ顧客・似た仕様でも、担当者によって提示価格が変わってしまう

`PrintSapiens`（印刷業界向けパッケージ）から **Excel/CSVで抽出したデータを活用する想定** で、
直接連携は行わず、既存システムを変更せずにスモールスタートできる構成をイメージしています。

AIが価格を自動決定するのではなく、**過去実績に基づく標準価格・推奨価格を提示し、最終判断は営業担当者が行う**設計です。

---

## 画面構成（タブ切り替え）

1. **ダッシュボード** — KPIカード／担当者別 価格傾向テーブル
2. **担当者別 価格分析** — 平均提示価格・標準価格との差分・受注率・粗利率・ばらつき度・AIコメント
3. **見積シミュレーション** — 条件入力 → 標準価格・推奨提示価格・低め/高め価格・営業経費上乗せ価格・価格ばらつきアラート・AIコメント
4. **同顧客の過去見積** — 顧客別の過去見積一覧と価格傾向サマリー・AIコメント
5. **類似案件検索** — 入力仕様に近い過去見積を類似度つきで一覧表示

---

## 使用技術

- React 18 + TypeScript（JSX/TSX）
- プレーンCSS（業務システム風 / ネイビー・ブルー・グレー基調）
- DB・バックエンド・外部AI APIは不要（フロントエンドのみ）
- サンプルデータはコード内に保持（`app.tsx` の `JOBS` から決定論的に生成）
- AIコメント・推奨価格・類似度は過去データをもとにロジックで擬似生成

ビルドツールを必要とせず動かせるよう、`@babel/standalone` でブラウザ上でTSX/JSXを変換する構成です。
（React・Babel は CDN から読み込みます）

## ローカルでの確認方法

`index.html` は外部ファイル（`app.tsx`）をブラウザから読み込むため、
`file://` で直接開くのではなく、簡易HTTPサーバ経由で表示してください。

```bash
# 例（任意の静的サーバでOK）
npx serve estimate-analysis-demo
# もしくは
python3 -m http.server   # 該当ディレクトリで実行
```

ブラウザで対象URLを開くと表示されます。

> ※ 本リポジトリのルールにより開発サーバ（`npm run dev` 等）の常時起動は行いません。
> 上記は表示確認用の静的配信のみです。

# 標準価格算出デモ（standard-price-demo）

印刷会社向けの **「標準価格のみ算出」デモ** です。
[print-estimate-demo](../print-estimate-demo/) のUI/UXをそのまま流用し、
見積条件を入力して **標準価格を算出する機能だけ** に絞ったシンプル版です。

**性格: 業務システム型**（印刷会社1社の社内業務を補助するシステムの提案デモ）

> ⚠️ 本デモで使用しているデータはすべて **モック（架空）データ** です。
> 実在する顧客名・社名・案件情報は含まれていません。

---

## 目的・想定ユースケース

- 「過去見積データをもとに、まずは **標準価格を出すだけ** の小さなデモが見たい」というニーズ向け
- DB・バックエンド不要。フロントエンドのみで動作

担当者別価格分析・同顧客の過去見積・類似見積検索など、より踏み込んだ分析は
全部入りの [print-estimate-demo](../print-estimate-demo/) 側で確認できます。

---

## 機能

残している機能：

- **見積条件入力**（顧客 / 印刷物種別 / サイズ / 用紙 / 色数 / 部数 / 加工 / 希望納期 / 担当者 / 営業経費率）
- **標準価格算出結果**（標準ベース価格 / 営業経費込み価格 / 価格レンジ / 標準価格比 / 想定原価 / 粗利率 など）
- **判断補助コメント**
- **処理ログ**

print-estimate-demo から省いた機能（このデモでは非表示）：

- 担当者別価格分析
- 類似見積検索（結果パネル・検索ボタン）

追加している機能：

- **管理画面**（プルダウン選択肢マスタ・単価マスタの管理。パスワード保護）

---

## 管理画面

見積画面右下の「⚙ 管理画面」リンクから `admin.html` にアクセスできます。

- **簡易認証**: パスワード入力欄でパスワードを入力しログインします（SHA-256 ハッシュ比較。平文はコード上に含まれません）。
- **プルダウン選択肢マスタ**: 印刷物種別・サイズ・用紙・色数・部数・加工・営業経費率の各プルダウンについて、選択肢の**追加・修正**ができます。
- **単価マスタ**: 印刷物種別ごとの基準単価（1部あたり）の**修正**ができます。
- 登録内容はブラウザの `localStorage` に保存され、リロードしても保持されます。
- **見積画面（`index.html`）の算出ロジックには反映されません。** `app.js` は print-estimate-demo と共有するビルド済みバンドルであり無改変のため、管理画面のマスタデータは独立した管理機能として動作します（画面内での追加・修正・保存・フィードバックはすべて実際に機能します）。

---

## 構成・実装方針

`print-estimate-demo` のビルド済みバンドル（`app.js`）を **そのまま複製** し、
UI/UX を一切変更しないために、ロジック・スタイルには手を加えていません。

不要な機能は、追加スクリプト [`customize.js`](./customize.js) が画面描画後に
該当パネル・ボタンを **非表示** にし、管理画面への導線を注入することで実現しています。

```
standard-price-demo/
├── index.html      … app.js → customize.js の順に読み込み
├── app.js          … print-estimate-demo と同一のバンドル（無改変）
├── customize.js    … 標準価格算出に不要なUIの非表示 / 管理画面リンクの注入
├── admin.html       … 管理画面（パスワード保護・プルダウン選択肢マスタ・単価マスタ）
└── README.md
```
